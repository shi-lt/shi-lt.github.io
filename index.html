git基本使用

发表于 2020-05-06 更新于 2020-06-03 分类于 git

常用命令

下载

  • git clone

    将github上的仓库克隆到本地磁盘并创建本地仓库

  • git pull

    将远程远程仓库拉取到本地仓库

上传

  • git add

    将本地文件添加到缓存区

  • git commit -m

    将缓存区内容提交到本地仓库（默认master分支）

  • git push -u origin

    将本地仓库内容推送到远程仓库（origin当前git服务器地址）

配置

  • git config –global user.name

    设置全局用户名

  • git config –global user.email

    设置全局邮箱

创建

  • git init

    初始化当前目录为一个本地仓库

  • git remote add origin url

    将创建的本地仓库与远程仓库进行关联

  • git branch

    创建分支

查看

  • git log

    查看提交记录

  • git branch

    查看分支（此命令还可进行分支创建、重命名、删除）

其他操作

  • git checkout

  • git mv

    移动文件

  • git rm

    删除文件

  • git merge

    合并分支
